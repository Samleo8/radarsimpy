radarsimpy.simulator¶

radarsimpy.simulator.run_simulator(radar, targets, noise=True)¶

Alias: radarsimpy.simulator()

Simulate baseband response of a radar with defined targets. Python engine.

Parameters

  • radar (Radar) – Radar model

  • targets (list[dict]) –

    Targets’ properties

    [

    {

    • location (1D array) -

      Location of the target (m), (x, y, z)

    • rcs (float) -

      Target RCS (dBsm)

    • speed (1D array) -

      Speed of the target (m/s), (vx, vy, vz). (default is (0, 0, 0))

    • phase (float) -

      Target phase (deg). (default is 0)

    }

    ]

  • noise (bool) – Flag to enable noise calculation

Returns

{

  • timestamp (3D array) -

    Timestamp for baseband data, [channels, pulses, adc_samples]

  • baseband (3D array) -

    Radar’s baseband output, [channels, pulses, adc_samples]

}

Return type

dict
