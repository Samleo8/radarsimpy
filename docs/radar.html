radarsimpy.Transmitter¶

class radarsimpy.Transmitter(fc, pulse_length, bandwidth=0, tx_power=0, repetition_period=None, pulses=1, slop_type='rising', channels=[{'location': (0, 0, 0)}])[source]¶

Bases: object

A class defines basic parameters of a radar transmitter

Parameters

  • fc (float or numpy.1darray) – Center frequency (Hz). If fc is a 1-D array, length should be equal to ‘pulses’

  • pulse_length (float) – Length of each pulse (s)

  • bandwidth (float) – Bandwith of each pulse (Hz)

  • tx_power (float) – Transmitter power (dBm)

  • repetition_period (float) – Pulse repetition period (s). ‘repetition_period’ >= ‘pulse_length’

  • pulses (int) – Total number of pulses

  • slop_type (str) – ‘rising’ or ‘falling’ slop for FMCW waveform

  • channels (list[dict] Properties of transmitter channels) –

    Properties of transmitter channels

    [

    {

    • location (numpy.1darray) -

      3D location of the channel <x. y. z> (m)

    • delay (float) -

      Transmit delay (s)

    • azimuth_angle (numpy.1darray) -

      Angles for azimuth pattern (deg)

    • azimuth_pattern (numpy.1darray) -

      Azimuth pattern (dB)

    • elevation_angle (numpy.1darray) -

      Angles for elevation pattern (deg)

    • elevation_pattern (numpy.1darray) -

      Elevation pattern (dB)

    • phase_code (numpy.1darray) -

      Phase code sequence for phase modulation (deg). If ‘chip_length’ == 0, or ‘chip_length’ is not defined, length of ‘phase_code’ should be equal to ‘pulses’

    • chip_length (float) -

      Length for each phase code (s). If ‘chip_length’ == 0, one pulse will have one ‘phase_code’. If ‘chip_length’ != 0, all ‘phase_code’ will be applied to each pulse

    }

    ]

Variables

  • channel_size (int) – Total number of transmitter channels

  • locations (numpy.2darray) – 3D location of the channel <x. y. z> (m)

  • az_angles (numpy.1darray) – Angles for azimuth patterns (deg)

  • az_patterns (numpy.1darray) – Azimuth pattern (dB)

  • el_angles (numpy.1darray) – Angles for elevation pattern (deg)

  • el_patterns – Elevation pattern (dB)

  • phase_code – Phase code sequence for phase modulation (deg). If ‘chip_length’ == 0, or ‘chip_length’ is not defined, length of ‘phase_code’ should be equal to ‘pulses’

  • chip_length (numpy.1darray) – Length for each phase code (s). If ‘chip_length’ == 0, one pulse will have one ‘phase_code’. If ‘chip_length’ != 0, all ‘phase_code’ will be applied to each pulse

  • delay (numpy.1darray) – Transmit delay (s)

  • wavelength (float) – Wavelength (m)

  • slope (float) – Waveform slope, ‘bandwidth’ / ‘pulse_length’
