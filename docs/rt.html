Raytracing¶

radarsimpy.rt.scene.scene()¶
radarsimpy.rt.lidar_scene.lidar_scene()¶
radarsimpy.rt.rcs.rcs_sbr()¶

Calculate target’s RCS by using raytracing

Parameters

  • model (str) – Path of the model

  • phi (float) – Observation angle phi (deg)

  • theta (float) – Observation angle theta (deg)

  • f (float) – Center frequency (Hz)

  • pol (list) – Polarization <x, y, z>

  • density (float) – Ray density (number of rays per wavelength)

Returns

Target’s RCS (m^2), use 10*log10(RCS) to convert to dBsm

Return type

float
